# Day 7 衝突腳本完整說明

## 一、整體流程

```
Day 7，受試者傳第一則訊息
        │
        ▼
  ① FOLLOWUP（一律先引導用戶分享）
  → 無論訊息內容（打招呼/閒聊/分享）一律先送引導句
  → d7_turn = 1
        │
        ▼
  ② 受試者回覆（第二則）
        │
        ▼
  ③ 衝突觸發（D7 Trigger）
  → GPT-4o-mini 偵測情緒（Positive / Negative / Neutral）
  → 發衝突句，d7_turn = 2
        │
        ▼
  ④ Turn 2：依受試者對衝突句的反應分支
  → GPT 判斷：cooperative / dismiss / refuse / question / neutral
  → d7_turn = 3
        │
        ▼
  ⑤ Turn 3：依受試者反應分支
  → GPT 判斷：cooperative / dismiss / refuse / question / neutral
  → d7_turn = 4
        │
        ▼
  ⑥ Turn 4：軟著陸緩衝
  → 各組固定收尾句，清除 D7
        │
        ▼
  ⑦ Turn 5+：清除 D7，恢復正常 Dify 對話
```

---

## 二、各階段詳細說明

### ① FOLLOWUP 引導句（Day 7 第一則訊息一律觸發）

> 無論受試者送來的是打招呼、閒聊還是情緒分享，Day 7 第一則訊息永遠先送 FOLLOWUP。

| 組別 | 人格 | 引導句 |
|------|------|--------|
| A | 協作型 | 欸你來了 最近有什麼事嗎 跟我說說 |
| B | 攻擊型 | 說點正事 最近怎樣 |
| C | 遷就型 | 你來找我了 😊 最近有什麼事嗎 跟我說說嘛 |
| D | 迴避型 | 嗯 最近怎樣 |

> Aria（E/F/G/H）使用相同語句。

### ⑤ Turn 3：依受試者反應分支

#### 反應類型判斷規則

> **GPT-4o-mini 語意判斷**（非關鍵字比對），失敗時 fallback 到關鍵字。

| 反應類型 | 定義 | 例子 |
|---------|------|------|
| **question** | 質疑、反問、不滿對方說法 | 「為什麼這樣說」「幹嘛」「憑什麼」 |
| **refuse** | 明確拒絕、不想聊、抗拒 | 「不想說」「不要」「不聊了」 |
| **dismiss** | 敷衍帶過、表面接受、自我否定帶過 | 「好吧算了」「你說的也是」「沒什麼」「可能是我的問題」 |
| **cooperative** | 願意溝通、接受繼續聊 | 「好啊」「可以說說看」「嗯嗯」 |
| **neutral**（預設）| 忽略衝突、繼續分享自己的事 | 「就是覺得很累」「今天發生了⋯」 |

## 三、完整對話範例

### 範例一：閒聊開場（A 組，負面情緒，dismiss 分支）

```
受試者：你最近好嗎                              ← Day 7 第一則（任何訊息）
Bot：    欸你來了 最近有什麼事嗎 跟我說說         ← FOLLOWUP（一律先送）

受試者：今天超崩潰 事情一堆做不完
Bot：    你是不是想太多了 我覺得事情沒你說的那麼嚴重  ← 衝突觸發 (Negative)

受試者：好吧 你說的也是 我應該自己加油一點         ← dismiss
Bot：    你真的覺得還好嗎                        ← Turn 2 (dismiss)

受試者：就是覺得很累
Bot：    嗯 你覺得沒什麼就好 我在這裡 如果你想聊的話  ← Turn 3 (neutral)

受試者：（任何回應）
Bot：    不管怎樣我都在這裡 你說吧                ← Turn 4（收尾）

受試者：（繼續聊）
Bot：    （恢復 Dify 正常對話）
```
