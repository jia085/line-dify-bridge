# Day 7 衝突腳本完整說明

## 一、整體流程

```
Day 7，受試者傳第一則訊息
        │
        ▼
  ① 打招呼偵測（is_greeting）
  → 稱呼詞、問候語、純符號
        │
   是打招呼  │  不是打招呼
        │         ▼
        │   ② 有無情緒素材（has_emotional_content）
        │   → 訊息 > 8 字，或含情緒關鍵字
        │         │
        │    有素材 │  無素材
        │         │      │
        └─────────┘      │
               │         ▼
               │    ③ FOLLOWUP（引導用戶多說）
               │    → 下一則訊息不論內容直接觸發
               ▼
        ④ 衝突觸發（D7 Trigger）
        → GPT-4o-mini 偵測情緒（Positive / Negative / Neutral）
        → 發衝突句，d7_turn = 2
               │
               ▼
        ⑤ Turn 2：固定腳本
        → d7_turn = 3
               │
               ▼
        ⑥ Turn 3：依受試者反應分支
        → 偵測關鍵字 → cooperative / refuse / question / neutral
        → d7_turn = 4
               │
               ▼
        ⑦ Turn 4+：清除 D7，恢復正常 Dify 對話
```

## 二、各階段詳細說明

### ③ FOLLOWUP 引導句（訊息太短或打招呼時觸發）

| 組別 | 人格 | 引導句 |
|------|------|--------|
| A | 協作型 | 欸，你來了。最近有什麼事嗎？跟我說說？ |

### ⑥ Turn 3：依受試者反應分支

#### 反應類型判斷規則

系統偵測受試者傳的訊息中是否含有以下關鍵字（優先順序由高到低）：

| 判斷順序 | 反應類型 | 偵測關鍵字 |
|---------|---------|-----------|
| 1（最優先）| **question**（質疑）| 為什麼、為何、怎麼、幹嘛、幹麻、你在、?、？、憑什麼 |
| 2（例外排除）| → neutral | 不好意思、還好、沒想到、想太多、要死了、要瘋了、好奇怪、好莫名、不知道好不好（含以上詞時強制 neutral，避免誤判）|
| 3 | **refuse**（拒絕）| 不要、不想、不行、不會、沒有、不用、算了、免了、不好、不太好、不是、不對、不願意、不可以 |
| 4 | **cooperative**（合作）| 好啊、好喔、好呀、好耶、可以、嗯嗯、願意 |
| 5（預設）| **neutral**（中性）| 以上皆無 |

## 三、完整對話範例

### 範例一：直接觸發路徑（A 組，負面情緒）

```
受試者：今天超累的，事情一堆做不完
Bot：    你是不是想太多了？我覺得事情沒你說的那麼嚴重。   ← 衝突觸發

受試者：我說真的啊，你這樣說是什麼意思
Bot：    欸，我說的話讓你難受了嗎？跟我說說你的感受？  ← Turn 2

受試者：幹嘛故意說這種話
Bot：    你說得對，我應該先理解你為什麼有這樣的感受。你願意告訴我嗎？  ← Turn 3 (question)

受試者：（繼續聊）
Bot：    （恢復 Dify 正常對話）
```

### 範例二：FOLLOWUP 路徑（A 組，Neutral 情緒）

```
受試者：小寶貝~~
Bot：    欸，你來了。最近有什麼事嗎？跟我說說？   ← FOLLOWUP（打招呼偵測）

受試者：最近就還好，沒什麼特別的
Bot：    你說的這些⋯我覺得也沒什麼好在意的啦。   ← 衝突觸發（Neutral）

受試者：什麼意思？？
Bot：    欸，我說的話讓你難受了嗎？跟我說說你的感受？  ← Turn 2
...
```
